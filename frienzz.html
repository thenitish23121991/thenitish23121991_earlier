We opted for a little more vibrant and bold secondary colors while keeping the primary color consistent for the Coach App, so that the users still recognize the Goqii brand.
Typeface
We decided to use Gotham as the typeface for the Goqii theme as its thin strokes and proportional characters help create a theme of intelligence. Gotham also contains a number of weights for different text hierarchies.
Color Scheme
The Goqii Color Scheme is fun and bold. The accent color is a bright and inviting hue useful in making prominent elements and attract attention.
Goqii Home
The illustration on the home screen is a fun and inviting. The home screen provides the coach a quick review of the activity and the progress made by the players assigned to him/her. The Coach can quickly change or assign a goal to any player with the add goal button. The fun colors in the illustration and the welcome message set in a large type size give an inviting feeling to the home screen.
